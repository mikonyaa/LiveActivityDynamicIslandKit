# Changelog

## 0.1.0

- Added reusable, domain-neutral Lock Screen and Dynamic Island SwiftUI surfaces.
- Added adaptive built-in styling and public light/dark appearance palettes.
- Added a polished Activity Lab with six recipes, app-owned ActivityKit attributes, safe exact-state deep links, and WidgetKit integration.
- Added recoverable, serialized local lifecycle handling with duplicate-start protection.
- Added finite progress normalization, valid timeline bounds, accessibility metadata, package tests, and demo policy tests.
- Documented the package boundary and migration path from pre-release `main`.
